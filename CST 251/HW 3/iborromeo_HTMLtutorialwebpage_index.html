Adding Content

HTML is an abbreviation for "Hyper Text Markup Language". This type of markup language helps authors publish websites that have structured components such as headers, lists, images, survey forms, and many more. Styles can be added to the website through CSS properties.


  • Here is how to insert an image:
  • img src="image.jpg" alt="image name"

    • Semantic Tags

    Using h1, h2, and h3 within a tag h1 is used for main headings, h2 for subheadings, h3 sub-subheadings, and so on. The tags range from h1 to h6. H1 is the highest section level when h6 is the lowest.

    • Here are some common semantic tag examples:
  • figure: adds a figure
  • form: adds a survey form
  • p: commonly used for paragraphs
  • li, ul, ol: used for bulletted and numbered lists

    • CSS Style Properties

    CSS is the abbreviation for "Cascading Style Sheets" where it is a way one can add styling to their website. This includes changing background color of text, figures, text color, font size and type, borders, and so on.


      Here are a couple examples of commonly used CSS tags:

  • color: changes font color
  • background-color: changes background color of text or body
  • font-size: changes font size
  • margin: adds margin space
  • text-align: aligns the text to center, left, or right side of the webpage.
